# 独立知识点动画生成项目 (Interactive Animator)

本项目是从 OpenMAIC 抽离并独立实现的一个交互式知识点动画课件生成系统。
它允许教师通过配置简单的"知识点要素"和"交互设计思路"，由经过 Prompt 工程深度调教的大模型现场编写结构化的、科学严谨的（支持KaTeX）、互动性强的前端代码，并且提供即时的**自然语言审校与重构修改（Refine）**体验，最终分发给学生。

---

## 核心特性

- **三大阶段自动管线**：1. 严谨的科学建模（提取公式/约束）；2. 交互式原生 JS+HTML 生成；3. KaTeX 公式依赖及 DOM 级生命周期劫持（后处理注入）。
- **分辨率全链路锁定 (1024×768)**：强制执行固定分辨率，废除自适应布局，确保生成内容的一致性与沉浸感。
- **状态化对齐策略**：已发布作品在 `Share` 页面水平居中；未发布作品在 `Review` 页面靠左，为右侧智脑面板留出充足操作空间。
- **智脑协同审阅面板 (AI Review Studio)**：全中文母语化界面，集成意图驱动对话、蓝图卡片化编辑与一键全量重构三大功能链路。
- **纯粹的前端沙盒**：生成代码均是独立、内聚、完全脱离外部打包器环境的单页面（无需 React），非常轻量安全。
- **端到端的开箱即用**：零配置的 SQLite 自动落盘建表；无需繁琐配置。

---

## 技术栈

- 框架：**Next.js 16 (App Router)**
- 持久层：**better-sqlite3** (本地生成 `animations.db`)
- 模型引擎优先级：
  1. **DeepSeek Pro** (`deepseek-v4-pro` - 128k) [首选]
  2. **Alibaba Qwen** (`qwen3-max` - 256k)
  3. **Google Gemini** (`gemini-3-flash-preview` - 1M+)
  4. **vLLM (Fallback)** (`gemma-4-moe` - 32k)
- 样式系统：**Tailwind CSS (V4)**
- 端口：**7001**

## API 路由

| 路径                             | 动作    | 描述 |
| -------------------------------- | ------- | ---- |
| `/api/animations`               | POST    | 创建草稿并在 DB 建立记录 |
| `/api/animations`               | GET     | 取得历史生成项目列表 |
| `/api/generate`                 | POST    | **核心端点**：触发两阶段 LLM 动画初始生成管线 |
| `/api/animations/[id]`          | GET/PUT | 查询实时进度（轮询），或更新记录（脚本字段） |
| `/api/generate/refine`          | POST    | 接收自然语言修改指令，下发带约束上下文的 LLM Refine 生成 |
| `/api/animations/[id]/publish`  | POST    | 修改最终状态为 Published |

## 审阅面板 (Review Panel) 详细说明

审阅面板 (`app/review/[id]/page.tsx`) 是本项目交互体验的核心，布局为**左侧演示 / 右侧智脑面板**不对称双分栏结构。

### 布局结构

```
┌────────────────────────────────────┬──────────────────────┐
│   左侧：1024×768 px 动画演示区        │  右侧：智脑协同面板      │
│   • 固定物理分辨率，靠左对齐          │  • 宽 500px            │
│   • iframe 沙盒实时预览              │  • 深空蓝配色 #0e111a   │
│   • 修改中显示量子同步遮罩            │  • 两大功能 Tab 入口    │
└────────────────────────────────────┴──────────────────────┘
```

### 智脑面板：对话快修 Tab

> **目标**：轻量、快速地收集局部修改意见并立即生效

1. **意图驱动入口（策略一）**
   - 用户首先从四个磁贴中选择意图类型：`视觉风格` / `功能交互` / `知识内容` / `其他问题`
   - 选中后输入框 placeholder、顶部提示文字、快捷指令分组均动态更新，消除认知断层

2. **意图指引输入框**
   - 支持多行自由输入
   - `Ctrl/Cmd + Enter` 快速发送
   - 发送按钮颜色跟随所选意图的主题色变化

3. **分组快捷指令（策略三）**
   - 按意图类型分组（视觉风格、功能交互、知识内容、其他问题）
   - 当前意图对应的组**自动展开**并标注"推荐"徽章
   - 其他分组默认折叠，可手动展开
   - **累加模式**：点击指令后以 `、` 追加到输入框，不会替换既有内容；再次点击已选指令可取消（右侧 `✓` 标识）

4. **修改历史**：最近 5 条操作记录，含时间戳，带紫色左边条

### 智脑面板：修改蓝图 Tab

> **目标**：对动画的底层结构脚本进行深度编辑，触发全量重建

1. **蓝图卡片化（策略六）**
   - 六张独立可展开卡片：`核心难点` / `教案大纲` / `场景布局` / `演员设定` / `交互逻辑` / `视觉特效`
   - 每张卡片含**状态徽章**（已完善 绿色 / 待填写 黄色）
   - 展开后显示内容预览，点击"编辑"进入内联编辑模式
   - 内联编辑支持多行文本，完成/取消均有明确按钮

2. **同步底座重构 (REGENESIS)**
   - **仅在修改蓝图 Tab 显示**，与对话快修的轻量 refine 链路严格解耦
   - 首先将所有卡片内容序列化为结构化 Markdown（`### 1. 场景布局 ...`）
   - 通过 `PUT /api/animations/{id}` 保存脚本，再触发 `/api/generate` 全量重建
   - 重建期间左侧演示区显示全屏遮罩动效

## 如何安装与启动

1. **克隆仓库**：
   ```bash
   git clone https://github.com/elibwudi/AInimation.git
   cd AInimation
   ```

2. **安装依赖**：
   本项目使用 `pnpm` 管理依赖，推荐使用 `pnpm`：
   ```bash
   pnpm install
   ```

3. **配置环境变量**：
   项目中已包含 `.env.example` 模板。请将其复制为 `.env` 并填写您的大模型 API Key（注意：**切勿将包含真实 Key 的 `.env` 文件提交到 GitHub**）：
   ```bash
   cp .env.example .env
   ```
   然后打开 `.env` 文件配置以下 API Key：
   ```env
   DEEPSEEK_API_KEY=您的_DeepSeek_API_Key
   DASHSCOPE_API_KEY=您的_阿里_DashScope_API_Key
   GEMINI_API_KEY=您的_Google_Gemini_API_Key
   ```

4. **启动服务**（运行在端口 `7001`）：
   * **方法 A（推荐，Windows 脚本）**：
     双击或在终端运行快速重启脚本：
     ```powershell
     .\restart.bat
     ```
     该脚本会自动清理 7001 端口上已存在的残留进程并拉起全新的 Next.js 开发服务器。
   * **方法 B（标准命令行）**：
     ```bash
     pnpm run dev
     ```

5. **浏览器访问**：
   在浏览器中打开 `http://localhost:7001` 即可开始使用！

## 设计约定（开发者须知）

| 约定 | 说明 |
|------|------|
| **分辨率锁定** | 所有 iframe 及生成的 HTML 均强制为 `1024×768px`，Prompt 模板中也明确注入此约束 |
| **未发布页靠左** | `review` 页的 iframe 容器使用 `justifyContent: flex-start` 以兼容右侧面板 |
| **已发布页居中** | `share` 页使用 `justifyContent: center` 给学生提供沉浸式观看体验 |
| **两条修改链路** | `refine` API（局部快修）与 `generate` API（全量重建）严格分离，不可混用 |
| **面板配色** | 深空蓝 `#0e111a` + 低饱和蓝紫 `#818cf8`，避免科幻青的视觉疲劳 |
| **快捷指令累加** | 点击快捷指令为追加模式（`、`连接），再次点击为取消，设计上鼓励多条组合提交 |
